This system is built using modern technologies such as Flask for web development, TensorFlow and Keras for deep learning, and OpenCV for image processing. The application integrates a trained EfficientNet model to ensure high accuracy in disease prediction.

About the System

Krushi Mitra – Crop Health Diagnostic System is an AI-powered web application designed to assist farmers in identifying plant diseases quickly and accurately. The system uses deep learning techniques to analyze leaf images and provide detailed insights including disease name, causes, and recommended treatments.

By simply uploading an image of a plant leaf, users can receive instant diagnosis powered by a trained machine learning model. The platform aims to reduce crop loss, improve productivity, and support farmers with accessible and intelligent decision-making tools.
